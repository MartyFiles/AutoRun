### 🧩 Что такое AutoRun?
Минималистичный, лёгкий и удобный менеджер автозагрузки для Windows.

#

### ⚙️ Что он умеет?
Работать с задачами автозагрузки из:
- Реестра
- Папки Startup
- Планировщика задач
- Служб

#

### 💸 Он бесплатный?
Да, версия **1.0 полностью бесплатна**.
